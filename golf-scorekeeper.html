Golf Scorekeeper App

Simple golf scoring for everyday golfers

Overview

The Golf Scorekeeper App is a personal project designed to make scorekeeping quick and easy on the course.

Current Features

  • Support for multiple golfers
  • Custom courses
  • 9-hole and 18-hole rounds
  • Automatic score totals
  • Mobile-friendly interface

Future Plans

  • Handicap tracking
  • Statistics dashboard
  • Cloud backup
  • Tournament mode
